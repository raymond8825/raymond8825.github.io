HammerSpoon教程

Hello World

所有好的编程教程都从某种 Hello World 示例开始，因此我们将使用 Hammerspoon 绑定键盘热键的能力来演示通过简单通知说 Hello World。在您的 init.lua 中放置以下内容：

hs.hotkey.bind({"cmd", "alt", "ctrl"}, "W", function()
  hs.alert.show("Hello World!")
end)

然后保存文件，单击 Hammerspoon 菜单栏图标并选择重新加载配置。您现在应该会发现按 ⌘+⌥+ctrl+W 将在屏幕上显示 Hello World 通知。

这里发生的是我们告诉 Hammerspoon 将匿名函数绑定到特定的热键。热键由修饰键表（在本例中为⌘、⌥ 和 ctrl）和普通键 (W) 指定。匿名函数只是一个没有名字的函数。我们可以用一个名字单独定义警报函数并将这个名字传递给 hs.hotkey.bind()，但是 Lua 可以很容易地定义内联函数。

Fancier Hello World

虽然 hs.alert 很有用，但您可能更喜欢使用 macOS 原生通知，您只需将前面的示例修改为：

hs.hotkey.bind({"cmd", "alt", "ctrl"}, "W", function()
  hs.notify.new({title="Hammerspoon", informativeText="Hello World"}):send()
end)

Spoon插件使用教程

Spoons

Spoons是HammerSpoon的官方预制插件

如何安装spoon?

spoon应该以 .zip 文件的形式分发。只需下载一个，解压缩（如果您的浏览器没有自动完成该部分），然后双击 Spoon。 Hammerspoon 将在 ~/.hammerspoon/Spoons/ 中为您安装它

如何使用Spoon?

这有两个步骤组成，加载Spoon并将其集成到您的配置中。默认 Spoon 在其主页或 ~/.hammerspoon/Spoons/NAME.spoon 中附带一些文档。在那里你应该找到 Spoon 提供的 API 的一些文档，以及它的任何特殊要求。

加载Spoon

对于大多数 Spoon，只需将 hs.loadSpoon("NAME") 添加到您的 Hammerspoon 配置中（注意 NAME 不应包含.spoon扩展名）。这将使Spoon在全局 Lua 命名空间中作为勺子spoon.NAME 可用。

加载 Spoon 后，您应该在使用它之前调用它的 start() 方法（如果它有start() 方法）。

请注意hs.loadSpoon()使用 package.path 来查找 Spoons。因此，您可以通过将这些路径添加到 package.path 来让它在其他路径中查找 Spoons，如下所示.如果您有正在开发的 Spoons，这将很有用。

-- Look for Spoons in ~/.hammerspoon/MySpoons as well
package.path = package.path .. ";" ..  hs.configdir .. "/MySpoons/?.spoon/init.lua"

集成到您的配置中

在大多数情况下，API 应该大致采用这种形式：

  • NAME:init() - 这由 hs.loadSpoon() 自动调用，并将执行所需的任何初始设置工作，但通常不应开始执行任何操作

  • NAME:start() - 如果需要任何类型的后台工作，此方法将启动它

  • NAME:stop() - 如果任何类型的后台工作正在运行，此方法将停止它

  • NAME:bindHotkeys(mapping) - 此方法用于告诉 Spoon 如何为其各种功能绑定热键。根据 Spoon，这些热键可能会立即绑定，或者在调用 :start() 时绑定。这个方法应该接受一个参数，一个表格的形式：

    { someFeature = {{"cmd", "alt"}, "f"},
    otherFeature = {{"shift", "ctrl"}, "b"}}

Spoon 还应该提供一些标准的元数据：

  • NAME.name - 包含勺子名称的字符串
  • NAME.version - 包含 Spoon 版本号的字符串
  • NAME.author - 包含勺子作者姓名/电子邮件的字符串
  • NAME.license - 包含有关适用于 Spoon 的许可证的一些信息的字符串，理想情况下包括许可证的 URL
  • (可选)NAME.homepage - 包含 Spoon 主页 URL 的字符串

许多 Spoons 会在这些基础上提供额外的 API 点，您应该查阅他们的文档以了解更多信息。
